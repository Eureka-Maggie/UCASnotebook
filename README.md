关于选课和学分  
校园生活
  - 校园墙 主要有两大平台，小程序“狐友”和校园集市（需加群），都需要身份认证才可互动
  - 食堂 西区有一食堂二食堂，东区有三食堂，好吃程度倒序排列
  - 外卖 雁栖湖比较偏，外卖主要是通过加群，骑手在固定几个时间配送怀柔县城的一些商家。此外还有一些附近的餐馆会建群自行配送，如满满家、邻家小馆、大铁牛螺蛳粉等
  - 讲座
  - 学生会
